◆ fetchImage()

uint32_t fetchImage ( const char * url,
uint8_t * dst_buf,
uint32_t max_bytes )

HTTP経由で指定されたURLから画像データを取得し、バッファに格納します。

  • 引数
    [in]url画像取得元のURL
  • 引数
    [in]dst_buf画像データを格納するためのバッファ（事前に確保済みであること）
  • 引数
    [in]max_bytes期待するデータサイズ（バイト数）
    戻り値
    uint32_t 実際に受信したデータのバイト数（失敗時は0）
  • 覚え書き
    期待するデータサイズ（max_bytes）と異なる場合でも、タイムアウトまたは データ終了まで受信を試みます。データ受信が継続している間は タイムアウトカウンタ（5000ms）がリセットされます。
